# Redfish API: /Oem/Kenmec/Config

## Overview

This document describes the Redfish API for Oem kenmec config.

## Endpoints

Config read:

```bash
GET /redfish/v1/ThermalEquipment/CDUs/1/Oem/Kenmec/Config.Read
```

Config write:

```bash
POST /redfish/v1/ThermalEquipment/CDUs/1/Oem/Kenmec/Config.Write
```
